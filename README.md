## Outline

This repository is boilerplate for React Web.

## Features

- TypeScript with FSA
- Webpack 4
- React 16
- Redux 4
- Recompose
- Prettier / ESLint
- Huskey / Lint-Staged
- commitizen


## Usage

```zsh
# package install
$ yarn

# develop mode
$ yarn start

# publish mode
$ yarn prod
```
